# Emma Troni — Portfolio

Personal portfolio showcasing my work in communication design, branding, editorial, and web design.

**[Live Site →](https://emmatroni.com)**

## About

I'm a Communication Design student at Politecnico di Milano with a focus on visual identities, typography, publishing, and UI/UX design. This portfolio collects selected projects and experiments.

## Built With

React · TypeScript · Vite · Tailwind CSS

## Highlights

- Custom page transitions with color morphing
- Scroll-driven text reveal animations
- Interactive image trail on cursor movement
- Fluid responsive typography
- Custom lerp-based cursor

## Local Development

```bash
npm install
npm run dev
```
